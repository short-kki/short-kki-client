[화면 요청: 메인 탭 - 홈 (숏폼 탐색)]

틱톡이나 릴스 같은 '풀스크린 버티컬 비디오' UI를 만들어줘.
실제 비디오 기능 대신, <View> 컴포넌트로 배경색을 깔아서 레이아웃을 잡아줘.

[요구사항]
1. 화면 전체를 꽉 채우는 비디오 영역 (회색 placeholder)
2. 우측 하단에 인터랙션 버튼 세로 배치:
   - 프로필 이미지 (원형)
   - 좋아요 (Heart Icon) + 숫자
   - ★중요: '큐에 담기' 버튼 (Download 혹은 Plus Icon) - Primary Color(#FF6B00)로 강조
   - 공유 (Share Icon)
3. 좌측 하단에 텍스트 정보 오버레이:
   - 레시피 제목 (굵고 큰 폰트)
   - 작성자 이름
   - 태그 (#초간단 #자취요리)
4. 하단 탭바(Bottom Tab)가 들어갈 공간을 고려해줘. (SafeAreaView 처리)
